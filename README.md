


# 🌦️ Weather App using API
A simple and responsive weather application that shows current weather and 5-day forecast using user input or GPS location.

![App Screenshot](https://cdn-icons-png.flaticon.com/256/4861/4861699.png)


---

## 🚀 Features

* 🔍 Search weather by city name
* 📍 Auto-detect user location (GPS)
* 🌡️ Current temperature display
* 📅 5-day weather forecast


---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* WeatherAPI

---



## 🔑 API Setup

This app uses WeatherAPI.

Replace API key in `script.js`:

```javascript
const API_KEY = "YOUR_API_KEY";
```

Get your API key from:
https://www.weatherapi.com/



---

## 📌 Usage

* Enter a city and click **Get Weather**
* OR click **Use My Location** for GPS-based weather
